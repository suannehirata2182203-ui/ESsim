# DIGI Website - Local Version

Локальная версия сайта DIGI с функциональным конфигуратором тарифов.

## Структура проекта

- `index.html` - Главная страница
- `tarif1/` - `tarif6/` - Страницы конфигурации тарифов
- `files/` - Статические ресурсы (CSS, JS, изображения)
- `images/` - Изображения (favicon и др.)

## Локальный запуск

```bash
python -m http.server 8000
```

Затем откройте в браузере: `http://localhost:8000`

## Деплой на Railway

Проект готов к деплою на Railway. Railway автоматически определит конфигурацию и запустит HTTP сервер.

## Функционал

- ✅ Полная копия оригинального сайта DIGI
- ✅ Интерактивный конфигуратор тарифов (6 тарифов)
- ✅ Автоматический расчет итоговой стоимости
- ✅ Интеграция с платежной системой
- ✅ Все трекеры и пиксели удалены
